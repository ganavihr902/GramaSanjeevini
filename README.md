🌿 GramaSanjeevini
📱 Project Overview

GramaSanjeevini is an Android-based rural healthcare application designed to improve access to medical information and healthcare services in rural areas. The app helps users search for medicines, view details, and access basic healthcare support using a simple and user-friendly interface.

🚀 Features
🔍 Medicine search functionality
💊 Detailed medicine information
👨‍⚕️ Basic healthcare guidance
📂 Organized and easy-to-use UI
🔐 User authentication (Login/Signup)
☁️ Firebase integration (if used in your project)
🛠️ Tech Stack
Language: Kotlin / Java
IDE: Android Studio
Database: Firebase / SQLite (based on your project)
Architecture: Activity-based structure
